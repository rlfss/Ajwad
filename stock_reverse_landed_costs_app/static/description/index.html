Cancel and Reset Landed Cost in Odoo

Reverse/Cancel WMS Landed Costs helps user to cancel Landed Costs which is already done/validated. This app is useful when user made mistake on landed cost and validate it. In Default Odoo after validate the landed cost there is no option to reverse or reset/change landed cost details. Use of this app user can easily cancel the validated landed cost and reset back to draft stage and able to change the details of the landed cost. When user cancel the landed cost it reverse all the calculation like it delete created journal entry, revert amount of the landed cost from the inventory valuation also it will reset the product cost price amount, so everything will be correct. Also this apps works on Odoo Enterprise and Community Both Edition.

Cancel Landed Cost.

User can cancel landed Cost, using cancel button.

Delete/Unlink journal entry

Journal entries will be automatically removed upon canceling the landed cost.

Set landed cost to draft

User can send landed cost to the draft, after canceling the Landed Cost

Module Working Flow


Description

Create Landed Cost

User creates and computes landed cost. Landed cost will be calculated for the products which is under selected transfer


Validate Landed Cost

User can validate after computing the additional landed cost.


Cancel Landed Cost

User has option to cancel the additional landed cost after validation, and journal entries, valuation adjustments and stock moves will be automatically removed.


Cancelled Landed Cost

User can see cancelled landed cost and set it to draft state.


Recompute Landed Cost

Here user has flexibilty to make any change, update value in landed cost recompute again.


Validate the landed Cost

User can validate the landed cost, and new stock move & new journal entry will be automatically created according to the updated values.
